## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

## Technology Stack

- **Framework**: [Next.js](https://nextjs.org) with React
- **Database**: SQLite with ESV Bible text
- **Styling**: Modern CSS with dark mode support
- **Font**: [Geist](https://vercel.com/font) font family

## Usage

1. **Navigate**: Use the sidebar to browse books and chapters
2. **Read**: View Bible text in the main reading pane
3. **Take Notes**: Use the notes panel on the right to add your thoughts
4. **Add References**: Click on any verse to add its reference to your current note
5. **Search**: Use the search bar to quickly find books and chapters
6. **Theme**: Toggle between light and dark modes using the theme button in the top bar
